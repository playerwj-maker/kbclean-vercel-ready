# KB클린 현장관리 앱

이 폴더가 GitHub / Vercel에 올라가야 하는 프로젝트 루트입니다.

필수 루트 파일:
- package.json
- index.html
- vite.config.js
- vercel.json
- src/App.jsx
- src/main.jsx
- src/index.css

로컬 실행:
```bash
npm.cmd install
npm.cmd run dev
```

Vercel 설정:
- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
